## ✨ Особенности

- 🔐 Анонимная авторизация без регистрации
- 🗝️ Приватные комнаты с паролями (хешируются)
- 💬 Текст, изображения, голосовые сообщения
- 🧹 Авто-очистка истории в 00:00
- 🌙 Тёмная/светлая тема с запоминанием
- ♿ Доступность: ARIA, клавиатурная навигация
- 📱 Адаптивный дизайн для мобильных
